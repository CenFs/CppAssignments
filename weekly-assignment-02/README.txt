a.What is the difference between a class and on object?
Class is a kind of abstraction, which includes actions and attributies of this object. It collect all objects which have same attributies and same actions.
Object is an instance of a class. Such as we can see all customs as a class, and custom A as an object. He has attributies like name, address, gender, etc., and actions like buy(), feedback(), add_to_cart(), etc..

b.What is public interface?
Public variables or functions can be accessed by its objects or other child classes.

c.What is private interface? 
Private variables or functions can be only accessed in its own class. Even its own objects cannot access its private variables or functions.
